# Magma — документация языка

Magma — компилируемый язык программирования с гибридным C-подобным и псевкодовым синтаксисом , предназначенный для написания программ используя минимальный набор конструкций.

---

## Запуск

```bash
PyMagm -o compiledfile file.magm
```

Если файл не указан, компилятор запускается в интерактивном режиме — можно вводить код прямо в терминале и завершить ввод сочетанием **Ctrl+D**.

---

## Типы данных

Magma поддерживает четыре базовых типа:

| Тип     | Описание                          | Пример              |
|---------|------------------------------------|----------------------|
| `int`   | Целое число                        | `int age = 18`       |
| `float` | Число с плавающей точкой           | `float pi = 3.14`    |
| `str`   | Строка                              | `str name = "Alex"`  |
| `bool`  | Логическое значение (`true`/`false`)| `bool ok = true`      |

### Объявление переменных

```magma
int score = 0
float rate = 1.5
str greeting = "hello"
bool active = true
```

Переменную можно объявить сразу с выражением справа:

```magma
int a = 5
int b = a + 10
```

### Присваивание

После объявления значение переменной можно менять без повторного указания типа:

```magma
int i = 1
i = i + 1
i = 20
```

> Переменную нельзя использовать в выражении до её объявления и нельзя присвоить значение необъявленной переменной — компилятор выдаст ошибку.

## Вывод — `write`

Выводит строку или значение переменной.

```magma
write("Hello, world!")
write(name)
```

Поддерживаются экранированные символы внутри строк, например `\n` (перенос строки) и `\t` (табуляция):

```magma
write("Line 1\nLine 2")
```

---

## Ввод — `read`

Считывает данные, введённые пользователем, и сохраняет их в переменную.

```magma
read(name)
```

Можно сразу вывести подсказку перед вводом:

```magma
read("Введите имя: ")
```

Если переменная ещё не объявлена, `read` создаёт её автоматически как строковую.

---

## Условия — `if`

```magma
int age = 1

if (age < 18) {
    write("Несовершеннолетний")
}
```

Условие может использовать любые сравнения и переменные, объявленные ранее.

---

## Циклы — `while`

```magma
int i = 0
while (i < 10) {
    write(i)
    i = i + 1
}
```

Тело цикла может содержать несколько инструкций на отдельных строках.

---

## Пустые операции — `pass` и `skip`

`pass` и `skip` — ничего не делающие инструкции. Используются как заглушки, когда синтаксически нужна инструкция, но логически делать пока нечего.

```magma
if (age < 18) {
    pass
}

while (running) {
    skip
}
```

`pass` и `skip` эквивалентны друг другу — оба означают "ничего не делать".

---

## Пример программы

```magma
str name = "мир"
write("Привет, ")
write(name)
write("!\n")

int counter = 0
while (counter < 5) {
    write(counter)
    write("\n")
    counter = counter + 1
}

bool done = true
if (done) {
    write("Готово!\n")
} else {
    pass
}
```

---

## Синтаксические правила

- Каждая инструкция располагается на отдельной строке.
- Блоки кода (`if`, `while`) заключаются в фигурные скобки `{ }`.
- Ключевые слова `true` и `false` зарезервированы для булевых значений.
- Имена переменных начинаются с буквы или `_` и могут содержать буквы, цифры и `_`.

---

## Обработка ошибок

Если в коде встречается синтаксическая ошибка (например, использование необъявленной переменной), компилятор выведет сообщение с указанием файла и проблемной строки, компиляция будет прервана.
